# KIS 계좌 상태 조회

## 스크립트 경로

프로젝트 루트의 `{{AGENT_DIR}}/scripts/` 에 있다. 실행 시 **절대 경로**를 사용한다.
먼저 `pwd`로 현재 디렉토리를 확인하고, 프로젝트 루트를 기준으로 경로를 구성한다.

- 인증 확인: `uv run <프로젝트루트>/{{AGENT_DIR}}/scripts/auth.py`
- API 조회: `uv run <프로젝트루트>/{{AGENT_DIR}}/scripts/api_client.py <서브커맨드>`

## 인자 처리

`$ARGUMENTS`를 파싱하여 조회 범위를 결정한다.

| 입력 | 서브커맨드 | 설명 |
|---|---|---|
| (인자 없음) | `balance` + `holdings` | 잔고와 보유종목 |
| `잔고`, `예수금`, `balance` | `balance` | 예수금/평가금액만 |
| `보유종목`, `종목`, `holdings` | `holdings` | 보유종목만 |
| `코스피`, `지수`, `index` | `index` | 코스피/코스닥 지수 |
| `전체`, `all` | `all` | 잔고 + 보유종목 + 지수 전부 |

## 실행 순서

### 1. 인증 확인

`auth.py`를 실행한다. `authenticated`가 `false`이면 "/auth 커맨드로 먼저 인증하세요"라고 안내하고 중단한다.

### 2. API 조회

`api_client.py <서브커맨드>`를 실행한다.

### 3. 결과 표시

JSON 결과를 사용자 친화적 표 형태로 정리한다.

**잔고 (balance) 표시 형식:**

| 항목 | 값 |
|---|---|
| 계좌 | XXXX**** (모의투자) |
| 예수금 | 78,903,041원 |
| 총평가 | 102,242,541원 |
| 평가손익 | +2,330,500원 |

**보유종목 (holdings) 표시 형식:**

| 종목 | 수량 | 평균단가 | 현재가 | 수익률 |
|---|---|---|---|---|
| SK하이닉스 | 15 | 808,466 | 894,000 | +10.58% |
| 삼성전자 | 10 | 153,300 | 190,000 | +23.94% |

**지수 (index) 표시 형식:**

| 지수 | 현재 | 등락 | 등락률 |
|---|---|---|---|
| 코스피 | 2,650.12 | +15.3 | +0.58% |
| 코스닥 | 850.45 | -3.2 | -0.37% |

## 주의사항

- 금액에 천 단위 콤마를 넣어 가독성을 높인다.
- 손익은 양수면 `+`, 음수면 `-` 부호를 명시한다.
- 계좌번호는 JSON의 `account` 필드를 그대로 사용한다 (이미 마스킹됨).
- 토큰, 앱키, 시크리트 등 민감 정보는 **절대 출력하지 않는다**.
- API 조회 결과의 JSON만 참조하며, config 파일을 직접 읽지 않는다.
